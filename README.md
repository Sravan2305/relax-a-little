## prerequisites

- vscode ^1.52.0

## installation

[install relax-a-little from the vs code marketplace](https://marketplace.visualstudio.com/items?itemName=open-devs.relax-a-little)

## development

- `npm install` or `yarn`
- in vs code, "Run", "Start Debugging" or press `F5`

## Publishing

- `npm run build`
- `vsce package`
- upload package to marketplace
